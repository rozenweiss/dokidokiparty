마비노기 모바일 길드 파티 매칭 툴 화면 기획서

1. 사용자 화면

1.1 사용자 화면 목적

사용자 화면은 길드원이 다음 작업을 수행하기 위한 영역입니다.

- 길드 전용 서비스 입장
- 대표 캐릭터명 입력
- 대표 캐릭터 하위 캐릭터 등록 및 수정
- 콘텐츠 확인
- 참여 캐릭터와 시간 선택
- 일반 또는 지원 신청
- 신청 내용 확인 및 수정
- 파티 매칭 결과 확인

사용자 화면에서는 자동 매칭 실행, 콘텐츠 설정, 직업별 역할 설정, 파티원 수동 이동 등의 관리자 기능을 제공하지 않습니다.

---

1.2 사용자 이용 흐름

1. 길드 공용 비밀번호 입력
2. 대표 캐릭터명 입력
3. 기존 대표 캐릭터 데이터 조회 또는 신규 생성
4. 하위 캐릭터 등록 및 정보 확인
5. 신청할 콘텐츠 선택
6. 참여할 하위 캐릭터 복수 선택
7. 참여 가능한 시작 시각 복수 선택
8. 일반 또는 지원 신청 선택
9. 신청 내용 확인 및 등록
10. 매칭 실행 후 결과 확인

---

1.3 길드 입장 화면

화면 목적

길드 외부 사용자의 접근을 제한하고 길드원만 서비스에 접속하도록 합니다.

표시 항목

- 길드명
- 안내 문구
- 길드 공용 비밀번호 입력란
- 입장 버튼

입력 항목

- 공용 비밀번호

동작

- 비밀번호가 일치하면 대표 캐릭터 입력 화면으로 이동
- 비밀번호가 일치하지 않으면 오류 문구 표시
- 일정 시간 동안 인증 상태 유지 가능

오류 문구 예시

- 비밀번호가 올바르지 않습니다.
- 관리자에게 공유받은 길드 비밀번호를 확인해주세요.

---

1.4 대표 캐릭터 입력 화면

화면 목적

대표 캐릭터명을 계정처럼 사용하여 사용자의 하위 캐릭터 정보를 불러옵니다.

대표 캐릭터의 의미

대표 캐릭터는 사용자를 구분하기 위한 계정 단위입니다.

대표 캐릭터에는 다음 정보만 저장합니다.

- 대표 캐릭터 닉네임

대표 캐릭터에는 다음 정보를 입력하지 않습니다.

- 직업
- 역할
- 전투력
- 마도 저항

대표 캐릭터 자체는 자동 매칭 대상이 아닙니다.

표시 항목

- 대표 캐릭터명 입력란
- 조회 버튼
- 신규 등록 버튼
- 최근 입력한 대표 캐릭터명
- 안내 문구

동작

기존 대표 캐릭터가 있는 경우

- 입력한 대표 캐릭터명과 일치하는 데이터를 조회
- 해당 대표 캐릭터의 하위 캐릭터 목록 표시
- 캐릭터 관리 또는 파티 신청 화면으로 이동

기존 대표 캐릭터가 없는 경우

- 등록된 정보가 없다는 안내 표시
- 신규 대표 캐릭터 생성 가능
- 생성 후 하위 캐릭터 등록 화면으로 이동

안내 문구 예시

대표 캐릭터명은 사용자 계정처럼 사용됩니다.
실제 파티에 참여할 캐릭터는 대표 캐릭터 아래에 별도로 등록해주세요.

---

1.5 하위 캐릭터 목록 화면

화면 목적

대표 캐릭터 아래에 등록된 실제 매칭 대상 캐릭터를 확인하고 관리합니다.

표시 항목

각 캐릭터 카드 또는 목록에 다음 정보를 표시합니다.

- 캐릭터 닉네임
- 직업
- 자동 지정 역할
- 기본 전투력
- 마도 저항
- 수정 버튼
- 삭제 버튼
- 활성화 상태

주요 버튼

- 캐릭터 추가
- 캐릭터 수정
- 캐릭터 삭제
- 파티 신청하기

빈 상태 문구 예시

등록된 캐릭터가 없습니다.
파티 신청에 사용할 캐릭터를 먼저 등록해주세요.

---

1.6 하위 캐릭터 등록 화면

화면 목적

실제로 파티 매칭에 사용할 캐릭터 정보를 등록합니다.

입력 항목

- 캐릭터 닉네임
- 직업
- 기본 전투력
- 마도 저항

자동 표시 항목

- 역할

역할은 사용자가 입력하거나 수정할 수 없습니다.

사용자가 직업을 선택하면 관리자 설정에 따라 다음 중 하나가 자동 표시됩니다.

- 탱커
- 서포터
- 딜러

직업 선택 방식

직업은 검색 가능한 드롭다운으로 제공합니다.

동작 방식:

1. 직업 입력 영역 선택
2. 전체 직업 목록 표시
3. 텍스트 입력 가능
4. 입력한 내용과 관련된 직업만 목록에 표시
5. 목록에서 직업 선택
6. 선택된 직업의 역할 자동 표시

직업은 직접 입력하여 저장할 수 없으며 관리자 등록 목록에서만 선택할 수 있습니다.

입력 검증

- 캐릭터 닉네임 필수
- 직업 선택 필수
- 기본 전투력 숫자 입력
- 마도 저항 숫자 입력
- 음수 입력 불가

버튼

- 저장
- 취소

---

1.7 하위 캐릭터 수정 화면

수정 가능 항목

- 캐릭터 닉네임
- 직업
- 기본 전투력
- 마도 저항
- 활성화 여부

직업을 변경하면 역할도 자동으로 변경됩니다.

확인 문구 예시

캐릭터 정보를 수정하시겠습니까?

이미 신청한 데이터에는 신청 당시 저장된 수치가 사용될 수 있습니다.

---

1.8 콘텐츠 목록 화면

화면 목적

현재 신청 가능한 콘텐츠를 확인합니다.

콘텐츠별 표시 항목

- 콘텐츠명
- 파티 인원
- 필요 마도 저항
- 신청 가능한 시작 시각 범위
- 시간 선택 간격
- 신청 가능 여부
- 신청하기 버튼

선택 표시 항목

필요에 따라 다음 항목도 사용자에게 보여줄 수 있습니다.

- 마도 압력
- 캐릭터별 예상 최종 전투력

상태 표시

- 신청 가능
- 신청 마감
- 관리자 매칭 진행 중
- 결과 확인 가능

---

1.9 파티 신청 화면

화면 목적

하위 캐릭터와 참여 가능한 시작 시각을 선택하여 파티 매칭을 신청합니다.

표시 정보

- 콘텐츠명
- 파티 인원
- 역할 구성
- 필요 마도 저항
- 마도 압력
- 선택 가능한 시작 시각

캐릭터 선택 영역

대표 캐릭터 하위의 모든 활성 캐릭터를 표시합니다.

캐릭터별 표시 정보:

- 선택 체크박스
- 캐릭터 닉네임
- 직업
- 역할
- 기본 전투력
- 마도 저항
- 해당 콘텐츠 최종 전투력
- 신청 가능 여부

필요 마도 저항 처리

콘텐츠 필요 마도 저항이 1 이상인 경우 캐릭터 마도 저항과 비교합니다.

미달 캐릭터:

- 마도 저항 부족 표시
- 부족 수치 표시
- 선택 체크박스 비활성화
- 신청 대상에서 제외

표시 예시:

마도 저항 부족
필요 1,600 / 현재 1,550 / 50 부족

콘텐츠 필요 마도 저항이 미입력 또는 0인 경우:

- 마도 저항 제한 미적용
- 모든 캐릭터 선택 가능

시간 선택 영역

관리자가 설정한 시작 시각과 시간 간격을 기준으로 선택지를 표시합니다.

예시:

- 20:00
- 20:30
- 21:00
- 21:30
- 22:00

여러 시간을 동시에 선택할 수 있습니다.

시간 일괄 적용 안내

선택한 모든 시간은 선택한 모든 캐릭터에 공통 적용됩니다.

예시:

선택 캐릭터:

- A
- B
- C

선택 시간:

- 20:00
- 21:00
- 22:00
- 23:00

모든 캐릭터 A, B, C는 네 개 시간의 매칭 후보가 됩니다.

일반 신청에서는 각 캐릭터가 최대 한 번만 배정되므로 최대 세 개 시간에 배정될 수 있습니다.

신청 유형

- 일반 신청
- 지원 신청

일반 신청 안내

- 선택한 각 캐릭터는 최대 한 번만 배정
- 동일 대표 캐릭터는 같은 시간에 한 캐릭터만 배정
- 캐릭터 수보다 시간이 많으면 일부 시간에는 배정되지 않을 수 있음

지원 신청 안내

- 부족한 역할과 빈자리 보완에 사용
- 동일 대표 캐릭터는 같은 시간에 한 캐릭터만 배정
- 서로 다른 시간에는 같은 캐릭터가 반복 배정될 수 있음

신청 버튼 활성화 조건

- 캐릭터 한 개 이상 선택
- 시간 한 개 이상 선택
- 신청 유형 선택
- 모든 선택 캐릭터가 필요 마도 저항 충족

---

1.10 신청 확인 화면

화면 목적

최종 등록 전에 신청 내용을 검토합니다.

표시 항목

- 대표 캐릭터명
- 콘텐츠명
- 신청 유형
- 선택 캐릭터 목록
- 캐릭터별 직업과 역할
- 캐릭터별 기본 전투력
- 캐릭터별 마도 저항
- 캐릭터별 최종 전투력
- 선택 시작 시각 목록

안내 문구

선택한 모든 캐릭터는 선택한 모든 시간의 매칭 후보로 등록됩니다.

동일 대표 캐릭터는 같은 시간에 한 캐릭터만 배정됩니다.

버튼

- 신청 완료
- 이전으로 돌아가기

---

1.11 신청 완료 화면

표시 항목

- 신청 완료 문구
- 콘텐츠명
- 신청 캐릭터 수
- 신청 시간 수
- 신청 유형
- 신청 내역 확인 버튼
- 콘텐츠 목록으로 이동 버튼

완료 문구 예시

파티 매칭 신청이 완료되었습니다.

관리자가 자동 매칭을 실행한 후 결과를 확인할 수 있습니다.

---

1.12 신청 내역 화면

화면 목적

현재 유효한 신청 내용을 확인하고 수정하거나 취소합니다.

신청별 표시 항목

- 콘텐츠명
- 대표 캐릭터명
- 선택 캐릭터
- 선택 시간
- 신청 유형
- 신청 시각
- 신청 상태

상태값 예시

- 신청 완료
- 매칭 대기
- 매칭 완료
- 결과 공개

사용자 기능

- 신청 수정
- 신청 취소

수정 시 처리

신청 캐릭터와 시간 선택을 다시 설정할 수 있습니다.

관리자가 이미 자동 매칭을 실행한 후에는 수정 또는 취소를 제한하거나, 기존 결과가 무효화될 수 있다는 안내가 필요합니다.

---

1.13 매칭 결과 화면

화면 목적

관리자가 공개한 파티 구성을 사용자에게 보여줍니다.

결과 분류

- 콘텐츠별
- 시작 시각별
- 파티 번호별

파티 카드 표시 항목

- 콘텐츠명
- 시작 시각
- 파티 번호
- 역할
- 캐릭터 닉네임
- 빈자리
- 부족 역할 및 인원

공개 정보

- 캐릭터 닉네임
- 역할

공개하지 않는 정보

- 대표 캐릭터명
- 직업
- 기본 전투력
- 최종 전투력
- 마도 저항
- 일반 또는 지원 신청 여부
- 전투력 균형 점수

표시 예시

21:00 / 파티 1

- 탱커: 캐릭터 A
- 서포터: 캐릭터 B
- 딜러: 캐릭터 C
- 딜러: 모집 중

부족 인원:

- 딜러 1명 부족

---

1.14 사용자 화면 내 주요 안내 문구

대표 캐릭터 안내

대표 캐릭터는 사용자 계정 역할을 하며 실제 파티에 자동 배정되지 않습니다.

캐릭터 선택 안내

선택하지 않은 캐릭터는 매칭 후보에 포함되지 않습니다.

시간 선택 안내

선택한 시간은 선택한 모든 캐릭터에 동일하게 적용됩니다.

중복 배정 안내

같은 대표 캐릭터에 등록된 캐릭터는 동일 시간에 한 캐릭터만 배정됩니다.

지원 신청 안내

지원 신청 캐릭터는 서로 다른 시간대에 반복 배정될 수 있습니다.

2. 관리자 화면

2.1 관리자 화면 목적

관리자 화면은 파티 매칭 서비스에 필요한 기준 데이터를 관리하고, 신청 현황을 확인하며, 자동 매칭을 실행하고 결과를 직접 수정하기 위한 영역입니다.

관리자는 다음 작업을 수행합니다.

- 길드 설정 관리
- 직업 및 역할 데이터 관리

2.19 재매칭 기능

처리 방식

초기화 대상:

- 파티
- 파티원 배정
- 관리자 수동 이동
- 관리자 수동 입력 캐릭터
- 빈자리
- 평균 전투력
- 미배정 목록
- 부족 인원 정보

최신 신청 데이터를 기준으로 다시 매칭하시겠습니까?

삭제 예정 시각

재매칭 실행 시각으로부터 48시간 후 신청 및 매칭 데이터를 삭제합니다.

2.20 데이터 삭제 관리

자동 삭제 기준

자동 매칭 실행 시각으로부터 48시간 후 삭제합니다.

삭제 대상

- 신청 데이터
- 선택 캐릭터 정보
- 선택 시간 정보
- 일반 및 지원 신청 구분
- 신청 당시 전투력 정보
- 매칭 파티
- 파티원
- 임시 캐릭터
- 미배정 목록
- 부족 인원
- 자동 매칭 실행 기록

유지 데이터

- 대표 캐릭터
- 하위 캐릭터
- 직업 목록
- 직업별 역할
- 콘텐츠 설정
- 길드 설정

관리자 표시 정보

- 자동 매칭 실행 시각
- 자동 삭제 예정 시각
- 남은 보관 시간

수동 삭제

관리자는 48시간이 지나기 전에도 신청 및 매칭 데이터를 수동 삭제할 수 있습니다.

확인 문구:

현재 신청 정보와 매칭 결과를 모두 삭제하시겠습니까?

삭제된 정보는 복구할 수 없습니다.
